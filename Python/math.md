# math

문제를 풀다보면 반올림, 올림, 내림이 필요할 때가 있다.

## 올림

```py
>>> import math #math 모듈을 먼저 import해야 한다.
>>> math.ceil(-3.14)    #결과는 -3
>>> math.ceil(3.14) #결과는 4
```

## 내림

```py
>>> import math
>>> math.floor(3.14)    #결과는 3
>>> math.floor(-3.14)   #결과는 -4
```

## 반올림

2번째 인자에 따라서 소수점 자리수를 정해줄 수 있다. 

```py
>>> round(3.1415)   #결과는 3
>>> round(3.1415, 2)   #결과는 3.14
```
