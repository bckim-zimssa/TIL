## 지역변수/ 전역변수

```python
cnt = 100

def func(n):
  lv = n + 1
  print(lv)
  cnt += 1 


# 여기서 n, lv 는 지역변수
# cnt 는 전역변수
```

```python
cnt = 100
def func():
  cnt = 0
  print(cnt)
# 지역변수 cnt/ 전역변수 cnt가 따로 있다.
>>> func()
0
>>> print(cnt)
100
```

```python
cnt = 100
def func():
  global cnt
  cnt = 0
  print(cnt)

>>> func()
0
>>> print(cnt)
0
```
