
# 레퍼런스 카운트

각각의 객체는 레퍼런스 카운트를 가지고 있다. 이 레퍼런스 카운트가 0이 되면 가비지 컬렉션에 의해서 메모리가 해제된다. 

```python
r = [1, 2, 3] # 레퍼런스 카운트 +1
s = r # +1
r = "hello" # -1
s =  "bye"  # -1  // 0
# 이제 [1,2,3] 리스트 객체는 소멸 대상이 된다.
```

## 파이썬 실행 원리

파이썬 소스 코드(.py) -> 바이트 코드로 변환 -> Python Virtual Machine(PVM)이 해석
